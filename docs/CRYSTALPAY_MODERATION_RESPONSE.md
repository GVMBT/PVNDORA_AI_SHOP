## 6. Подробное описание деятельности проекта, товаров и услуг

**PVNDORA** — специализированный сервис цифровой дистрибуции, предоставляющий доступ к премиум-подпискам на ведущие AI-сервисы и цифровые платформы.

### Основная деятельность:
- Продажа лицензионных ключей и инвайт-ссылок для SaaS-платформ
- Предоставление готовых аккаунтов с активными подписками
- Реселлерство премиум-доступов к AI-инструментам

### Категории товаров и услуг:

1. **AI-сервисы для работы с текстом:**
   - ChatGPT Plus / Pro
   - Claude Pro (Anthropic)
   - Perplexity Pro
   - Character.AI Premium

2. **AI-сервисы для генерации изображений:**
   - Midjourney Premium
   - Stable Diffusion (коммерческие лицензии)
   - DALL-E API доступы

3. **Инструменты для разработчиков:**
   - GitHub Copilot (индивидуальные и бизнес-подписки)
   - JetBrains All Products Pack
   - VS Code Premium расширения

4. **Образовательные подписки:**
   - Студенческие аккаунты (Edu-версии)
   - Триальные периоды для тестирования
   - Шеринг-аккаунты (Shared access)

5. **API-доступы:**
   - OpenAI API ключи
   - Anthropic API ключи
   - Другие AI-платформы API

### Бизнес-модель:
- **Мгновенная цифровая доставка** — товары выдаются автоматически после оплаты
- **Гарантия валидности** — все ключи и аккаунты проверяются перед продажей
- **Поддержка клиентов** — консультации по выбору продукта и помощь в настройке
- **Реферальная программа** — система вознаграждений для постоянных клиентов

### Технические особенности:
- Telegram Mini App для удобной покупки через мессенджер
- Веб-интерфейс для браузерной версии
- Автоматизированная система выдачи товаров
- Многоязычный интерфейс (RU, EN, UK, DE, FR, ES, TR, AR, HI)
- Поддержка множественных валют (USD, RUB, EUR, UAH, TRY, INR, AED)
